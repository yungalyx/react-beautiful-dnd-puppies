<h2> How to run </h2>

After extracting from zip file or cloning the repo, navigate the project folder with command lines 
 
    cd Desktop
    cd puppy-app
    npm install 
    npm start

it will take a little while(~30sec) to build. (node and their dependecies, not me!)
